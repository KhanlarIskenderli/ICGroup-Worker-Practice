# ICGroup Worker Practice

## Кейс-задача №3

Данный репозиторий содержит решение кейс-задачи по производственной практике,
проходившей в компании ICGroup.AZ (Азербайджан).

## Описание задачи

Разработана программа учета сотрудников (WORKER), включающая:

- ввод данных о сотрудниках;
- хранение данных в коллекции;
- расчет стажа работы;
- вывод сотрудников со стажем выше заданного значения;
- обработку случая отсутствия подходящих сотрудников.

## Используемый язык

Встроенный язык 1С (адаптированное решение).

## Структура проекта

- Worker.bsl — описание структуры сотрудника
- Main.bsl — основная логика программы
